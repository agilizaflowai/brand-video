Observações de produção

Compatibilidade testada em: Gmail (web/mobile), Outlook 365 web/desktop, Apple Mail, iOS Mail. A estrutura em <table> com estilos inline é obrigatória porque o Outlook desktop descarta <style> blocks e CSS externos.

Fonte da marca: Fraunces (serif). Como o Outlook não carrega web fonts, o fallback é Georgia → Times New Roman → serif genérico. Em clients que carregam a web font (Apple Mail, Gmail web), a marca aparecerá em Fraunces corretamente.

Instalação (Gmail): abrir Configurações → ver todas as configurações → "Assinatura" → colar todo o bloco HTML da variante desejada. Habilitar "Inserir assinatura antes do texto citado".

Instalação (Outlook): Arquivo → Opções → Correio → Assinaturas → criar nova → colar HTML. No Outlook desktop, abrir o editor em HTML (não texto formatado).

Substituir: [Nome do Partner], partner@magno.capital, telefone. Manter estrutura visual.
